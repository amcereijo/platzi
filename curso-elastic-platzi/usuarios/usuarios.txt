POSTMAN
 -> crear con indice propio
  PUT http://localhost:9200/usuarios/_doc/1
    { "nombre": "nombre", "apellido": "surname" }
-> crear con indice generado
  POST http://localhost:9200/usuarios/_doc/
  { "nombre": "Name surname" }

-> Crear con archivo en modo bulk
  POST http://localhost:9200/usuarios/_bulk
    binay => usuarios.json

-> Busqueda en indice
  GET http://localhost:9200/usuarios/_search

-> Busqueda en indice con id 1
  GET http://localhost:9200/usuarios/_doc/1 => con metadatos
  GET http://localhost:9200/usuarios/_source/1 => sin metadatos

-> Busqueda en indice campos
  GET http://localhost:9200/usuarios/_mapping
